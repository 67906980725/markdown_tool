# markdown tool py

``` shell
# 安装 python3 (3.10/3.11)
cd md_tool_py
python3 -m venv .venv # 创建虚拟环境
source ./venv/bin/activate # 激活虚拟环境-unix 
# .\ENV_DIR\Scripts\activate # 激活虚拟环境-windows
pip install -r ./requirements.txt # 安装依赖

# 如果安装依赖时出现如下报错可以尝试更换国内源或移除国内源
# ERROR: Could not find a version that satisfies the requirement beautifulsoup4 (from versions: none)
# ERROR: No matching distribution found for beautifulsoup4
```
